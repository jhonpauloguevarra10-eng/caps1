## Message Routing: Chat Example

```
User A sends: "Hello everyone!"

1. Client A emits Socket.IO event:
   socket.emit('chat-message', {
     roomId: "ROOM123",
     message: "Hello everyone!",
     username: "Alice"
   })

2. Server receives event in handler:
   socket.on('chat-message', (data) => {
     // Validate
     // Get user from users map
     // Broadcast to room
   })

3. Server broadcasts to entire room:
   io.to(roomId).emit('receive-message', {
     username: "Alice",
     message: "Hello everyone!",
     timestamp: new Date(),
     socketId: socket.id
   })

4. All clients in room receive:
   socket.on('receive-message', (data) => {
     uiManager.addChatMessage(
       data.username,      // "Alice"
       data.message,       // "Hello everyone!"
       data.timestamp      // timestamp
     )
   })

5. UI updates:
   - Create <div> for message
   - Shows: "Alice: Hello everyone! 10:30 AM"
   - Adds to chat panel
   - Scrolls to latest
   - Updates notification badge if panel closed
```
